# Projects

Each project should live in its own folder under `projects/`.

## Structure

- `shared/`: reusable CSS and JS for project pages
- `<project-slug>/`: one self-contained page per project
- `<project-slug>/assets/media/`: paper PDF, preview image, video, and other page media
- `<project-slug>/assets/figures/`: exported figures for the project page
