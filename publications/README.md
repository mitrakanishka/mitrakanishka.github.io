# Publications

Each publication should live in its own folder under `publications/`.

## Structure

- `shared/`: reusable CSS and JS for publication / project pages
- `<publication-slug>/`: one self-contained page per publication
- `<publication-slug>/assets/media/`: paper PDF, preview image, video, and other page media
- `<publication-slug>/assets/figures/`: exported figures for the project page
